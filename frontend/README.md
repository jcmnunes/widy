This folder contains the frontend code of the Widy app. It is a React/Redux SPA.

## Available npm scripts

To start the development server:

```
$ npm start
```

To build the site:

```
$ npm run build
```

To test the build by starting a local HTML server:

```
$ npm run serve
```

To run unit-tests:

```
$ npm run tests
```

To run unit-tests with coverage report:

```
$ npm run coverage
```

To run ESLint:

```
$ npm run lint
```

To run Prettier:

```
$ npm run format
```
