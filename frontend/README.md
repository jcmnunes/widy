This is the frontend repo of the Widy app. It is a React/Redux SPA. The backend
repo can be found [here](https://github.com/jcmnunes/widy-backend).

<img src="https://raw.githubusercontent.com/jcmnunes/widy-frontend/master/src/images/eod_screenshot.png" width="100%">

Widy is a productivity app. It has three main features:

1. It tracks your daily work.
2. It helps you stay focused.
3. It generates handy reports.

UI components are from the [UI-Capsules kit](https://uic.binarycapsule.tech/).

## Available npm scripts

To start the development server:

```
$ npm start
```

To build the site:

```
$ npm run build
```

To test the build by starting a local HTML server:

```
$ npm run serve
```

To run unit-tests:

```
$ npm run tests
```

To run unit-tests with coverage report:

```
$ npm run coverage
```

To run ESLint:

```
$ npm run lint
```

To run Prettier:

```
$ npm run format
```
